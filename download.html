How to get MockQA

Either clone from GitHub

If you're comfortable with the command line and have Git installed, execute the following commands in order:

$ cd /path/to/yourproject
$ git clone git://github.com/mennovanslooten/mockqa.git

Or download a ZIP

If you'd rather dowload a ZIP file with the latest version, simply click this button: Download ZIP After downloading, extract the ZIP and copy the mockqa directory into your project.
